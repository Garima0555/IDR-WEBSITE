# IDR-WEBSITE
A responsive landing page for the IDR platform built using HTML and CSS, featuring modern UI design, smooth scrolling navigation, hover effects, and interactive sections like About, Pillars, Community, Pipeline, and Contact.
This project is a responsive landing page designed for the IDR platform.

Features:
- Modern UI layout
- Responsive design
- Smooth scrolling navigation
- Hover animations
- Floating icons animation
- Pipeline ripple animation
- Community and contact sections

Technologies Used:
HTML5
CSS3
